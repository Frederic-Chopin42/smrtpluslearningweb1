Common Keyboard Shortcuts

1. Copy
    a. Ctrl + C (Windows)
    b. Cmd + C (Mac OS)
2. Cut
    a. Ctrl + X (Windows)
    b. Cmd + X (Mac OS)
3. Paste
    a. Ctrl + V (Windows)
    b. Cmd + V (Mac OS)
4. Quit application
    a. Alt + F4 (Windows)
    b. Cmd + Q (Mac OS)
5. Undo 
    a. Ctrl + Z (Windows)
    b. Cmd + Z (Mac OS)

Document Keyboard Shortcuts

1. Symbols
    a. Alt + (# - Different For Every Symbol) (Windows)
    b. Option + Letter keys (Mac OS)
2. Delete line of text
    a. Cmd + delete (Mac OS)
3. Delete a word
    a. Option + delete (Mac OS)
    b. Highlight a line of text
    c. Shift + → / ← 
    d. Cmd + → / ← 
4. Highlight Words
    a. Ctrl  + Shift + → /← (Windows)
    b. Opt  + Shift + → /← (Mac OS)
5. Move the cursor to the beginning of the next word
    a. Ctrl + → (Windows)
    b. Opt + → (Mac OS)
6. Move the cursor to the beginning of the previous word.
    a. Ctrl + ← (Windows)
    b. Opt + ← (Mac OS)
7. Move the cursor to the beginning of the next paragraph.
    a. Ctrl + ↓ (Windows)
    b. Move the cursor to the beginning of the previous paragraph
    c. Ctrl + ↑ (Windows)
